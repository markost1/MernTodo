MERN To-Do App

A full-stack MERN application for managing notes and tasks with a clean, responsive UI. Built to demonstrate modern React + Node.js + MongoDB skills.

Features

Create, Read, Update, Delete (CRUD) notes

Responsive design with modern UI

Toast notifications for actions

Ready for future enhancements like authentication and search

Tech Stack

Frontend: React, React Router, Axios, Tailwind CSS

Backend: Node.js, Express

Database: MongoDB (Mongoose)

Quick Start
# Clone repo
git clone https://github.com/markost1/mern-todo.git

# Install backend
cd backend
npm install
npm run dev

# Install frontend
cd ../frontend
npm install
npm run dev


Visit http://localhost:5173 to start using the app.
